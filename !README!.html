Regarding update from versions prior to 2.0

Version 2.0 of BeDrive is esentially an entirely new application, this means
updating from older versions is more difficult and has more possible risks.

If updating from version prior to 2.0, we highly recommend doing the following:

  1. Backup BeDrive database.
  2. Move old BeDrive files to a different folder (don't delete them).
  3. Upload new files into an empty folder.

Afterwards you can copy user uploaded files (if stored locally) and move
.env file from old folder, see documentation for more details on this.
